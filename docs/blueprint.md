# **App Name**: Aesthetiq CRM

## Core Features:

- Client Profile Management: Maintain detailed client profiles including personal information, medical history, consent forms, and before/after photos. Track demographics for enhanced segmentation.
- Treatment History: Log comprehensive treatment history, including treatment details, products used, technician notes, and client feedback. Associate treatments with client profiles automatically.
- Appointment Scheduling: Schedule and manage appointments with automated reminders and confirmations. Allow clients to book appointments through a web portal. Automatically check for conflicts.
- Treatment Plan Tracking: Define and track detailed treatment plans with multiple sessions, products, and pre/post-care instructions. Adjust plans based on client progress and feedback. The tool should be able to decide when to suggest adjustments.
- Automated Workflow via n8n Integration: Configure automated workflows for client follow-ups, marketing campaigns, and internal notifications using n8n integration. The workflows should allow triggering personalized SMS messages or emails.
- Inventory Management: Track inventory of products used in treatments with alerts for low stock. Automatically update the number of products in stock.
- Analytics and Reporting: Generate reports on key performance indicators, client demographics, treatment effectiveness, and revenue. The tool should be able to automatically detect meaningful patterns in data.

## Style Guidelines:

- Primary color: Soft mint green (#B8E7C9) to evoke feelings of cleanliness and freshness.
- Background color: Light gray (#F0F0F0), a desaturated and bright tint of the primary color, to provide a clean and neutral backdrop.
- Accent color: Pale cyan (#A0CED9), an analogous color, used sparingly to draw attention to interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for a modern, readable interface.
- Use minimalist, line-style icons in a darker shade of mint green for visual clarity.
- Employ a clean, spacious layout with clear visual hierarchy for easy navigation.
- Use subtle transitions and animations to enhance user experience and provide feedback on interactions.
